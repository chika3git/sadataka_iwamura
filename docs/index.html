初代三重県令・佐賀の賢人

岩村 定高

佐賀に生まれ、激動の明治維新を駆け抜け、
三重の礎を築いた官僚。
その生涯と残された記録を紐解く。

近代化への情熱

天保6年（1835年）、肥前国佐賀に生まれる。佐賀藩士として義祭同盟に参加し、明治維新後は新政府に出仕。

北海道開拓判官などを経て、明治9年（1876年）に初代三重県令に就任。伊勢暴動の鎮圧や地租改正、産業の近代化に手腕を振るい、今日の三重県の基礎を作り上げた。

主な活動領域

佐賀・維新期

佐賀藩士として尊王攘夷運動に奔走。大隈重信らと共に新しい時代の扉を開いた。

三重県令時代

県政の統合・安定化に尽力。道路整備や製糸業など産業振興を推進した。

中央官僚時代

貴族院議員、農商務省の要職を歴任し、晩年まで国政に寄与し続けた。
